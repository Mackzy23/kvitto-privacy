Integritetspolicy för Kvitto

Senast uppdaterad: 2026-05-31

Vilka uppgifter samlar vi in?

Kvitto samlar in följande uppgifter:

— E-postadress (vid registrering)

— Bilder på kvitton (bearbetas tillfälligt av AI, lagras ej)

— Extraherad kvittoinformation (butik, datum, artiklar, priser)

Hur använder vi dina uppgifter?

Dina uppgifter används enbart för att tillhandahålla appens funktioner — spåra utgifter och visa historik. Vi säljer aldrig dina uppgifter till tredje part.

AI-bearbetning

Kvittobilder skickas till Anthropic Claude API för textextraktion. Bilderna lagras inte permanent — endast den extraherade texten sparas.

Datalagring

Dina kvitton lagras säkert i Supabase med kryptering. Du kan när som helst radera ditt konto och all tillhörande data via appens inställningar.

Dina rättigheter (GDPR)

Du har rätt att:

— Begära ut dina uppgifter

— Rätta felaktiga uppgifter

— Radera ditt konto och all data

— Invända mot behandling

Kontakt

Vid frågor, kontakta oss på: din@email.com

Ändringar

Vi kan uppdatera denna policy. Fortsatt användning av appen innebär att du godkänner eventuella ändringar.
